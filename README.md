KenZen AI Finance Dashboard

A dark-theme personal finance dashboard built with Streamlit.  
It shows real-time spending analytics, savings insights, anomaly detection, and clean data visualizations.

 Features
- Live account balance & spending summary  
- Spending by category  
- Daily spending trends (Plotly)  
- Automatic anomaly detection  
- KenZen AI insights (budget, savings, patterns)  
- Recent transactions table  
- Fully responsive dark UI

 Tech Stack
- Python  
- Streamlit  
- Pandas  
- NumPy  
- Plotly
